# BotSale

Стартовая реализация Telegram-бота на `aiogram 3.x` + `SQLAlchemy` для каталога продаж с тремя типами выдачи:

- `icc_id_chat`
- `bot_chat`
- `auto_delivery`

## Что реализовано в коде

- каркас модульной архитектуры (`handlers/services/repositories/models/config`)
- основные SQLAlchemy-модели по ТЗ (users, roles, catalog, products, orders, stock, ui)
- репозитории для каталога и заказов
- сервисы для:
  - выбора работников на смене
  - загрузки и безопасной выдачи стока (`with_for_update(skip_locked=True)`)
- базовые пользовательские/worker-хендлеры
- сидер стартовых категорий и товаров с ценами из ТЗ
- скрытая категория `SECONDARY_GU` (не отображается в витрине)

## Запуск

1. Создайте `.env` и укажите:
   - `BOT_TOKEN`
   - `DATABASE_URL`
   - `REDIS_URL`
2. Установите зависимости:
   ```bash
   pip install -e .
   ```
3. Запустите:
   ```bash
   python -m app.main
   ```
